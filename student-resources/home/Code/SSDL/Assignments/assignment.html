Some assignments

Using SSDL...

  1. Write a program that displays a graphic that moves back and forth from left to right.

  2. In the previous program, draw a line from the lower-left corner of the screen to the moving graphic.

  3. Write a program that allows you to move a graphic in all directions using the WASD keys.

  4. Write a program that displays a square made out of lines. Make each line a different color.

  5. In the previous program, make the square grow when the spacebar is pressed.

  6. Still in the previous program, make the square shrink when some other key is pressed.

  7. Write a program that draws lines from the point (250,500) to the points (x,0), where x is all of the points in the following array: int xValues[] = {0, 5, 10, 30, 31, 50, 55, 100, 200, 220, 330, 490, 495, 500}; (Use a loop)

  8. Change the previous program such that it only shows one of the lines at a time.
